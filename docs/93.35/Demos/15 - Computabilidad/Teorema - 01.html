Composición de funciones computables es computable:

Sean , todas funciones computables

Demo

Como y son computables , existen programas que computan a dichas funciones.

Gracias a esto puedo usar macros para representar a dichas secuencias de instrucciones:

xxxxxxxxxx
Z1 <- g1(X1, ..., Xn)
Z2 <- g2(X1, ..., Xn)
...
Zk <- gk(X1, ..., Xn)
Y <- f(Z1, Z2, ..., Zk)

Entonces, como puedo hacer un programa que compute a la función , esta es computable.
