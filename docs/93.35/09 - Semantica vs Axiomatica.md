|        Semantica         |       Axiomatica        |
| :----------------------: | :---------------------: |
| $\alpha$  es tautologia  |   $\alpha$ es teorema   |
|  $\alpha \in C(\emptyset)$  | $\emptyset \vdash \alpha$  |
|  $\alpha \in C(\Gamma)$  | $\Gamma \vdash \alpha$  |
| $\Gamma$ es satisfacible | $\Gamma$ es consistente |
